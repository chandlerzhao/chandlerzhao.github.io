# 如何在树莓派上安装Kali Linux渗透测试系统
准备工作：
> 1、硬件：我使用的是树莓派3 Model B、  
    8G 空白microSD卡一张、
    HDMI to VGA转接器一个
    USB键盘、鼠标各一个

> 2、软件：
   Kali linux arm版镜像，[官网](https://www.kali.org/downloads/)上有下载，
   armhf、armel版本的区别请看[这里]（http://www.veryarm.com/872.html）
   Win32DiskImager 以管理员身份运行
